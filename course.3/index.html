  • border - with the default of 10px dotted gray

  • text-indent - this paragraph starts with an indentation of 50px
  • text-transform - we capitalize this entire line
  • width and height - this paragraph has a hardcoded width of 800px and height of 300px
  • overflow - this page shows overflow if the render window is too small by default
  • text-overflow - this line shows text ellipsis for a small enough line
